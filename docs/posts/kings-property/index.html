The property

Let us explore one of the most useful symmetry ideas in definite integration. If \(f\) is integrable on \([a,b]\), King’s property says

\[ \boxed{\int_a^b f(x)\,dx=\int_a^b f(a+b-x)\,dx.} \]

In simple language: reflecting the graph horizontally across the midpoint \(x=(a+b)/2\) does not change its signed area over the interval.

Adding the two equal forms gives the useful identity

\[ \boxed{\int_a^b f(x)\,dx =\frac12\int_a^b\bigl[f(x)+f(a+b-x)\bigr],dx.} \]

Important: the property preserves signed area, so it remains valid even when parts of the graph lie below the \(x\)-axis.

Let me prove it mathematically

Begin with the reflected integral

\[ I=\int_a^b f(a+b-x)\,dx. \]

Now use the substitution

\[ u=a+b-x,\qquad du=-dx. \]

The limits reverse:

\[ x=a\Longrightarrow u=b, \qquad x=b\Longrightarrow u=a. \]

Therefore

\[ \begin{aligned} I &=\int_b^a f(u)(-du)\\ &=\int_a^b f(u)\,du\\ &=\int_a^b f(x)\,dx. \end{aligned} \]

In the last line, the dummy variable \(u\) has simply been renamed as \(x\). This proves King’s property.

Now let us see the graphical proof

Focus on the transformation

\[ x\longmapsto a+b-x. \]

It reflects every point across the midpoint \(m=(a+b)/2\). Imagine a very narrow strip at \(x=t\) under \(y=f(x)\). Its approximate signed area is

\[ f(t)\,\Delta x. \]

After reflection, its partner lies at \(x=a+b-t\). The reflected curve has height

\[ f\bigl(a+b-(a+b-t)\bigr)=f(t). \]

Reflection also preserves the width \(\Delta x\). Therefore, the two strips have exactly the same signed area. Pairing every strip in this way shows that the two complete integrals must be equal.

The complete code behind the interactive graph is included below. Expand the block to study, copy, or adapt it.

NoteComplete JavaScript source 
(() => {
  const host = document.getElementById("king-svg");
  const slider = document.getElementById("king-position");
  if (!host || !slider) return;

  const NS = "http://www.w3.org/2000/svg";
  const a = 0, b = 6, W = 900, H = 470;
  const pad = { left: 58, right: 24, top: 28, bottom: 48 };
  const plotW = W - pad.left - pad.right;
  const plotH = H - pad.top - pad.bottom;
  const yMax = 3.15;
  const f = x => 1.35 + 0.42 * Math.sin(1.55 * x) + 0.16 * x;
  const reflected = x => f(a + b - x);
  const X = x => pad.left + (x - a) / (b - a) * plotW;
  const Y = y => pad.top + (yMax - y) / yMax * plotH;
  const base = Y(0);
  const els = {
    t: document.getElementById("king-t"),
    x: document.getElementById("king-x"),
    rx: document.getElementById("king-rx"),
    height: document.getElementById("king-height"),
    areas: document.getElementById("king-areas")
  };

  const svg = document.createElementNS(NS, "svg");
  svg.setAttribute("viewBox", `0 0 ${W} ${H}`);
  svg.setAttribute("preserveAspectRatio", "xMidYMid meet");
  svg.setAttribute("aria-hidden", "true");
  host.appendChild(svg);

  const node = (name, attrs = {}, parent = svg) => {
    const el = document.createElementNS(NS, name);
    Object.entries(attrs).forEach(([key, value]) => el.setAttribute(key, value));
    parent.appendChild(el);
    return el;
  };
  const text = (value, x, y, attrs = {}) => {
    const el = node("text", { x, y, ...attrs });
    el.textContent = value;
    return el;
  };
  const points = (fn, lo = a, hi = b, count = 320) => {
    let d = "";
    for (let i = 0; i <= count; i++) {
      const x = lo + (hi - lo) * i / count;
      d += `${i ? "L" : "M"}${X(x).toFixed(2)},${Y(fn(x)).toFixed(2)} `;
    }
    return d;
  };
  const areaPath = (fn, lo, hi) => {
    const curve = points(fn, lo, hi, 150).replace(/^M/, "L");
    return `M${X(lo)},${base} ${curve}L${X(hi)},${base} L${X(lo)},${base} Z`;
  };
  const integral = (fn, lo, hi, n = 600) => {
    if (hi < lo) return -integral(fn, hi, lo, n);
    const h = (hi - lo) / n;
    let sum = 0.5 * (fn(lo) + fn(hi));
    for (let i = 1; i < n; i++) sum += fn(lo + i * h);
    return sum * h;
  };

  const grid = node("g", { stroke: "#aebdcb", "stroke-opacity": ".18" });
  for (let i = 0; i <= 6; i++) {
    node("line", { x1: X(i), y1: pad.top, x2: X(i), y2: base }, grid);
    text(String(i), X(i), base + 27, { fill: "#cbd8e4", "text-anchor": "middle" });
  }
  for (let y = 0; y <= 3; y++)
    node("line", { x1: pad.left, y1: Y(y), x2: W - pad.right, y2: Y(y) }, grid);

  node("line", { x1: pad.left, y1: base, x2: W-pad.right, y2: base, stroke: "#aebdcb" });
  node("line", { x1: X(3), y1: pad.top, x2: X(3), y2: base,
    stroke: "#ffffff", "stroke-opacity": ".35", "stroke-dasharray": "7 7" });
  text("midpoint", X(3), pad.top + 14, { fill: "#cbd8e4", "text-anchor": "middle" });
  text("f(x)", pad.left + 10, pad.top + 20, { fill: "#7be1eb", "font-weight": "700" });
  text("f(a+b−x)", pad.left + 10, pad.top + 42, { fill: "#ffd27a", "font-weight": "700" });

  const blueArea = node("path", { fill: "#57b9c5", "fill-opacity": ".22" });
  const goldArea = node("path", { fill: "#f3c56b", "fill-opacity": ".22" });
  node("path", { d: points(f), fill: "none", stroke: "#57b9c5", "stroke-width": "4" });
  node("path", { d: points(reflected), fill: "none", stroke: "#f3c56b", "stroke-width": "4" });
  const blueStrip = node("rect", { fill: "#7be1eb" });
  const goldStrip = node("rect", { fill: "#ffd27a" });
  const bluePoint = node("circle", { r: 7, fill: "#7be1eb" });
  const goldPoint = node("circle", { r: 7, fill: "#ffd27a" });

  function draw() {
    const t = Number(slider.value);
    const rt = a + b - t;
    const left = Math.min(t, rt), right = Math.max(t, rt);
    blueArea.setAttribute("d", areaPath(f, a, left));
    goldArea.setAttribute("d", areaPath(reflected, right, b));

    const setStrip = (rect, point, x, fn) => {
      const width = 10;
      rect.setAttribute("x", X(x) - width / 2);
      rect.setAttribute("y", Y(fn(x)));
      rect.setAttribute("width", width);
      rect.setAttribute("height", base - Y(fn(x)));
      point.setAttribute("cx", X(x));
      point.setAttribute("cy", Y(fn(x)));
    };
    setStrip(blueStrip, bluePoint, t, f);
    setStrip(goldStrip, goldPoint, rt, reflected);

    els.t.textContent = `t = ${t.toFixed(2)}`;
    els.x.textContent = `x = ${t.toFixed(2)}`;
    els.rx.textContent = `x = ${rt.toFixed(2)}`;
    els.height.textContent = `${f(t).toFixed(3)} = ${reflected(rt).toFixed(3)}`;
    els.areas.textContent =
      `${integral(f, a, left).toFixed(3)} ≈ ${integral(reflected, right, b).toFixed(3)}`;
  }

  function setFromPointer(event) {
    const rect = svg.getBoundingClientRect();
    const viewX = (event.clientX - rect.left) / rect.width * W;
    const ratio = Math.max(0, Math.min(1, (viewX - pad.left) / plotW));
    slider.value = String(a + ratio * (b-a));
    draw();
  }

  let dragging = false;
  svg.addEventListener("pointerdown", event => {
    dragging = true;
    svg.setPointerCapture(event.pointerId);
    setFromPointer(event);
  });
  svg.addEventListener("pointermove", event => dragging && setFromPointer(event));
  svg.addEventListener("pointerup", () => dragging = false);
  svg.addEventListener("pointercancel", () => dragging = false);
  slider.addEventListener("input", draw);
  draw();
})();

Verifying the picture with Python

The same asymmetric function can be plotted and integrated numerically with Python. This gives an independent computational check of the graphical argument.

Show the Python code 
import numpy as np
import matplotlib.pyplot as plt

plt.style.use("dark_background")

a, b = 0, 6
x = np.linspace(a, b, 1000)

def f(x):
    return 1.35 + 0.42 * np.sin(1.55 * x) + 0.16 * x

y_original = f(x)
y_reflected = f(a + b - x)

area_original = np.trapezoid(y_original, x)
area_reflected = np.trapezoid(y_reflected, x)

print(f"Area under f(x):       {area_original:.8f}")
print(f"Area after reflection: {area_reflected:.8f}")

fig, ax = plt.subplots(figsize=(9, 5), facecolor="#0c1521")
ax.set_facecolor("#0c1521")
plt.plot(x, y_original, label=r"$f(x)$", linewidth=2.5)
plt.plot(x, y_reflected, label=r"$f(a+b-x)$", linewidth=2.5)
plt.fill_between(x, y_original, alpha=0.18)
plt.axvline((a + b) / 2, color="gray", linestyle="--", label="midpoint")
plt.xlabel("x")
plt.ylabel("y")
plt.title("A function and its reflection")
plt.legend()
plt.grid(alpha=0.2)
plt.tight_layout()
plt.show()
Area under f(x):       11.51982490
Area after reflection: 11.51982490

The two printed values agree up to numerical rounding. Python confirms what the substitution already proved exactly.

Why the averaged form is powerful

Let

\[ I=\int_a^b f(x)\,dx. \]

King’s property gives another expression for the same number:

\[ I=\int_a^b f(a+b-x)\,dx. \]

Adding and dividing by \(2\) gives

\[ I=\frac12\int_a^b\left[f(x)+f(a+b-x)\right]dx. \]

This is most useful when the sum inside the brackets simplifies.

Let me work through an example

Evaluate

\[ I=\int_0^{\pi/2}\frac{\sin x}{\sin x+\cos x}\,dx. \]

Here \(a+b=\pi/2\). King’s property gives

\[ I=\int_0^{\pi/2} \frac{\cos x}{\cos x+\sin x}\,dx. \]

Adding the two equations,

\[ 2I=\int_0^{\pi/2}1\,dx=\frac{\pi}{2}. \]

Hence

\[ \boxed{I=\frac{\pi}{4}}. \]

A more powerful JEE-style example

Evaluate

\[ J=\int_0^\pi \frac{x}{1+\sin x}\,dx. \]

The factor \(x\) makes direct integration look unpleasant. Reflecting \(x\) across \([0,\pi]\) replaces it by \(\pi-x\), while \(\sin(\pi-x)=\sin x\). Thus

\[ J=\int_0^\pi \frac{\pi-x}{1+\sin x}\,dx. \]

Adding this to the original form gives

\[ 2J=\pi\int_0^\pi\frac{dx}{1+\sin x}. \]

Now rationalise the denominator:

\[ \frac{1}{1+\sin x} =\frac{1-\sin x}{\cos^2x} =\sec^2x-\sec x\tan x. \]

Although the two terms are individually improper at \(x=\pi/2\), their difference has a finite continuous limit there. An antiderivative on each side is \(\tan x-\sec x\), and the combined definite integral equals \(2\). Therefore

\[ 2J=2\pi \qquad\Longrightarrow\qquad \boxed{J=\pi}. \]

The important move was not the final integration. King’s property first removed the troublesome factor \(x\).

Takeaway

King’s property tells us that a definite integral is unchanged when its interval is reflected. Algebraically, reversing the limits cancels the minus sign in \(du=-dx\). Geometrically, every signed-area strip has an equal reflected partner. The slider makes these two descriptions visible as the same idea.
