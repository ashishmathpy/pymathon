The problem that starts our investigation

Let

A={1,a1,a2,…,a18,77} A=\{1,a_1,a_2,\ldots,a_{18},77\}

be a set of integers satisfying

1<a1<a2<⋯<a18<77. 1<a_1<a_2<\cdots<a_{18}<77.

Define

A+A={x+y:x,y∈A}. A+A=\{x+y:x,y\in A\}.

If A+AA+A contains exactly 3939 elements, find

a1+a2+⋯+a18. a_1+a_2+\cdots+a_{18}.

NoteQuestion source

This question appeared in JEE (Main) 2022, 28 June, Morning Shift (Shift 1), Mathematics.
View the original-question listing

At first, the information seems insufficient. We know only the first and last elements of AA, not the eighteen integers in between. The number 3939 is the clue: it is the smallest possible size of A+AA+A. A set whose sumset is this small must have a very special structure.

By the end of the article, we will prove that structure rather than guess it. The final answer will be

702. \boxed{702}.

Difficulty: Intermediate
Prerequisites: Sets, inequalities, and arithmetic progressions
Main idea: Few distinct pairwise sums mean many sums coincide. Those coincidences force equal consecutive gaps.

1. What is a sumset?

If AA and BB are sets of numbers, their sumset is

A+B={a+b:a∈A,b∈B}. A+B=\{a+b:a\in A,\ b\in B\}.

The notation A+AA+A means that both addends are chosen from the same set. Repetition is allowed when choosing the two addends: a+aa+a is included. However, the result is a set, so a value is written only once even if several pairs produce it.

A small example

Take

A={1,3,5}. A=\{1,3,5\}.

The addition table is

+ 1 3 5
1 2 4 6
3 4 6 8
5 6 8 10

Therefore,

A+A={2,4,6,8,10}, A+A=\{2,4,6,8,10\},

and |A+A|=5|A+A|=5. The vertical bars mean “number of elements”.

There are 32=93^2=9 ordered choices of two elements, but only five distinct answers. For example,

1+5=3+3=5+1. 1+5=3+3=5+1.

Such equalities are called collisions of sums.

Now compare this with B={1,3,8}B=\{1,3,8\}:

B+B={2,4,6,9,11,16}. B+B=\{2,4,6,9,11,16\}.

Thus |B+B|=6|B+B|=6. The irregular gap from 33 to 88 creates an additional distinct sum. This hints at the principle we will prove:

Equally spaced sets have the smallest possible sumsets.

2. Ordering gives a guaranteed chain of sums

Let a set of nn distinct real numbers be written in increasing order:

A={x1,x2,…,xn},x1<x2<⋯<xn. A=\{x_1,x_2,\ldots,x_n\}, \qquad x_1<x_2<\cdots<x_n.

Every number below belongs to A+AA+A:

x1+x1<x1+x2<⋯<x1+xn<x2+xn<x3+xn<⋯<xn+xn. \begin{aligned} x_1+x_1 &<x_1+x_2 <\cdots <x_1+x_n\\ &<x_2+x_n <x_3+x_n <\cdots <x_n+x_n. \end{aligned}

Why is the middle inequality strict? Since x1<x2x_1<x_2,

x1+xn<x2+xn. x_1+x_n<x_2+x_n.

The chain contains

n+(n−1)=2n−1 n+(n-1)=2n-1

different sums. Consequently:

ImportantSumset lower bound

For every finite set AA of nn distinct real numbers,

|A+A|≥2n−1. \boxed{|A+A|\geq 2n-1.}

This proof needs only ordering. In particular, it works for sets of integers, rational numbers, or real numbers.

What the theorem does—and does not—say

The theorem does not claim that A+AA+A always has exactly 2n−12n-1 elements. It says that 2n−12n-1 is the minimum possible.

For n=3n=3:

{1,3,5}+{1,3,5}={2,4,6,8,10} \{1,3,5\}+\{1,3,5\} =\{2,4,6,8,10\}

has 2(3)−1=52(3)-1=5 elements, while

{1,3,8}+{1,3,8}={2,4,6,9,11,16} \{1,3,8\}+\{1,3,8\} =\{2,4,6,9,11,16\}

has six.

3. Arithmetic progressions attain the lower bound

Suppose AA is an arithmetic progression:

A={a,a+d,a+2d,…,a+(n−1)d},d>0. A=\{a,a+d,a+2d,\ldots,a+(n-1)d\}, \qquad d>0.

Choose the terms with indices ii and jj, where 0≤i,j≤n−10\leq i,j\leq n-1. Their sum is

(a+id)+(a+jd)=2a+(i+j)d. (a+id)+(a+jd)=2a+(i+j)d.

Only the value of i+ji+j matters. Its possible values are

0,1,2,…,2n−2, 0,1,2,\ldots,2n-2,

which gives exactly 2n−12n-1 distinct sums. In fact,

A+A={2a,2a+d,2a+2d,…,2a+(2n−2)d}. A+A =\{2a,2a+d,2a+2d,\ldots,2a+(2n-2)d\}.

So A+AA+A is itself an arithmetic progression, with the same common difference.

For example,

A={1,5,9,13} A=\{1,5,9,13\}

gives

A+A={2,6,10,14,18,22,26}. A+A=\{2,6,10,14,18,22,26\}.

Here |A|=4|A|=4 and |A+A|=7=2(4)−1|A+A|=7=2(4)-1.

4. The equality case: minimum size forces an AP

We have shown one direction:

A is an AP⇒|A+A|=2n−1. A\text{ is an AP}\quad\Longrightarrow\quad |A+A|=2n-1.

The surprising part is the converse.

ImportantMinimal-sumset theorem

For a finite set A={x1<x2<⋯<xn}A=\{x_1<x_2<\cdots<x_n\} of real numbers, with n≥2n\geq2,

|A+A|=2n−1⇔A is an arithmetic progression. \boxed{|A+A|=2n-1 \quad\Longleftrightarrow\quad A\text{ is an arithmetic progression}.}

We now prove the difficult direction in small steps.

Step 1: The master chain already fills the sumset

Recall the 2n−12n-1 sums

x1+x1<x1+x2<⋯<x1+xn<x2+xn<x3+xn<⋯<xn+xn. \begin{aligned} x_1+x_1 &<x_1+x_2<\cdots<x_1+x_n\\ &<x_2+x_n<x_3+x_n<\cdots<x_n+x_n. \end{aligned} \tag{1}

If |A+A|=2n−1|A+A|=2n-1, there is no room for any additional value. Therefore every sum xi+xjx_i+x_j must equal one of the values in this master chain.

Step 2: Insert sums made with x2x_2

Consider

x2+x1<x2+x2<⋯<x2+xn. x_2+x_1<x_2+x_2<\cdots<x_2+x_n. \tag{2}

Its endpoints are already in the master chain:

x2+x1=x1+x2 x_2+x_1=x_1+x_2

and x2+xnx_2+x_n is the value immediately after x1+xnx_1+x_n.

Between these endpoints, the master chain contains exactly

x1+x3,x1+x4,…,x1+xn, x_1+x_3,\ x_1+x_4,\ldots,x_1+x_n,

which are n−2n-2 values. The second chain also contains exactly n−2n-2 interior values:

x2+x2,x2+x3,…,x2+xn−1. x_2+x_2,\ x_2+x_3,\ldots,x_2+x_{n-1}.

There are no other sumset values available. Both lists are strictly increasing, so their terms must match in order:

x2+xi=x1+xi+1,i=2,3,…,n−1. x_2+x_i=x_1+x_{i+1}, \qquad i=2,3,\ldots,n-1.

Rearranging gives

xi+1−xi=x2−x1,i=2,3,…,n−1. x_{i+1}-x_i=x_2-x_1, \qquad i=2,3,\ldots,n-1.

The same equality is automatically true for i=1i=1. Thus

x2−x1=x3−x2=⋯=xn−xn−1. x_2-x_1=x_3-x_2=\cdots=x_n-x_{n-1}.

All consecutive gaps are equal, which is precisely the definition of an arithmetic progression. This completes the proof.

The central intuition

A randomly spaced set creates many distinct sums. To reach the absolute minimum 2n−12n-1, every sum outside the guaranteed master chain must collide with a sum already in it. The first row of necessary collisions forces

x2+xi=x1+xi+1, x_2+x_i=x_1+x_{i+1},

and therefore forces every consecutive gap to be the same.

5. Solving the original problem

The set

A={1,a1,a2,…,a18,77} A=\{1,a_1,a_2,\ldots,a_{18},77\}

contains

1+18+1=20 1+18+1=20

elements. Hence the general lower bound is

|A+A|≥2(20)−1=39. |A+A|\geq2(20)-1=39.

We are told that |A+A|=39|A+A|=39, so equality occurs. By the minimal-sumset theorem, AA must be an arithmetic progression.

Let its common difference be dd. There are 1919 equal gaps from the first term to the twentieth term, so

77=1+19d. 77=1+19d.

Therefore,

19d=76⇒d=4. 19d=76 \quad\Longrightarrow\quad d=4.

Thus

A={1,5,9,13,…,73,77}, A=\{1,5,9,13,\ldots,73,77\},

and the requested terms are

a1,a2,…,a18=5,9,13,…,73. a_1,a_2,\ldots,a_{18}=5,9,13,\ldots,73.

Their sum is

∑i=118ai=182(5+73)=9⋅78=702. \begin{aligned} \sum_{i=1}^{18}a_i &=\frac{18}{2}(5+73)\\ &=9\cdot78\\ &=\boxed{702}. \end{aligned}

There is also a quick symmetry check. The sum of all 2020 terms is

202(1+77)=780. \frac{20}{2}(1+77)=780.

Removing the two endpoints gives

780−1−77=702. 780-1-77=\boxed{702}.

6. Explore sumsets with Python

The mathematics proves the result for every valid set. Python is useful for experimenting with examples, seeing collisions, and checking conjectures.

A basic sumset tool

def sumset(A, B=None):
    """Return the sorted sumset A+B; use B=A when B is omitted."""
    A = set(A)
    B = A if B is None else set(B)
    return sorted(a + b for a in A for b in B)


A = {1, 3, 5}
B = {1, 3, 8}

print("A + A =", sumset(A))

Find which pairs collide

Hide code 
from collections import defaultdict


def sum_representations(A):
    """Map each sum to unordered pairs that produce it."""
    values = sorted(set(A))
    representations = defaultdict(list)

    for i, x in enumerate(values):
        for y in values[i:]:
            representations[x + y].append((x, y))

    return dict(sorted(representations.items()))


for total, pairs in sum_representations({1, 3, 5}).items():
    print(f"{total:2} can be made by {pairs}")

This prints, among other lines,

Test the theorem on examples

Hide code 
def is_arithmetic_progression(A):
    values = sorted(set(A))
    if len(values) < 3:
        return True

    d = values[1] - values[0]
    return all(
        values[i + 1] - values[i] == d
        for i in range(len(values) - 1)
    )


def report(A):
    values = sorted(set(A))
    n = len(values)
    size = len(sumset(values))
    print("A =", values)
    print(f"|A| = {n}, |A+A| = {size}, lower bound = {2*n - 1}")
    print("Arithmetic progression?", is_arithmetic_progression(values))


report({1, 5, 9, 13})
print()

Search all small sets

The next experiment lists every four-element subset of {0,1,…,8}\{0,1,\ldots,8\} whose sumset has the minimum size 77.

Hide code 
from itertools import combinations

n = 4
minimum = 2 * n - 1
minimal_sets = []

for candidate in combinations(range(9), n):
    if len(sumset(candidate)) == minimum:
        minimal_sets.append(candidate)

print("Sets with a minimum-size sumset:")
for candidate in minimal_sets:
    print(candidate)

print(
    "Every result is an AP:",
    all(is_arithmetic_progression(A) for A in minimal_sets),
)
Sets with a minimum-size sumset:
Every result is an AP: True

This is experimental evidence, not a substitute for the proof. A finite search checks only the cases inside its chosen range; the theorem covers all finite sets of real numbers.

7. A broader viewpoint

This problem is a first glimpse of additive combinatorics, the study of how the arithmetic structure of a set is reflected in sets such as A+AA+A and A−AA-A.

The quantity

|A+A||A| \frac{|A+A|}{|A|}

is called the doubling constant of AA. A small doubling constant suggests that AA is highly structured. In our extreme one-dimensional case,

|A+A|=2|A|−1 |A+A|=2|A|-1

forces the strongest possible structure: an arithmetic progression.

You do not need additive combinatorics to solve the original question. Still, it is useful to see the larger mathematical theme:

The number of distinct sums reveals how regularly the original set is arranged.

8. Common mistakes

Counting pairs instead of distinct sums

A+AA+A is a set of values, not a list of pairs. Although there are n2n^2 ordered pairs (x,y)(x,y), different pairs may give the same sum.

Forgetting that an element may be used twice

Because xx and yy are independently chosen from AA, sums such as 2x=x+x2x=x+x are allowed.

Assuming the AP structure without proving equality

The lower bound |A+A|≥2n−1|A+A|\geq2n-1 alone does not make AA an AP. We need the given equality |A+A|=2n−1|A+A|=2n-1 and the equality-case theorem.

Miscounting the number of terms

The set in the problem has 2020 terms:

1,18 intermediate terms,77. 1,\quad 18\text{ intermediate terms},\quad 77.

Therefore it has 1919 consecutive gaps, not 2020.

Conclusion

For a set of nn distinct ordered numbers, the chain formed from the smallest and largest elements guarantees at least 2n−12n-1 distinct pairwise sums. Arithmetic progressions attain this bound, and attaining the bound forces all consecutive gaps to be equal.

In the original problem, |A|=20|A|=20 and |A+A|=39=2|A|−1|A+A|=39=2|A|-1. Hence AA must be

{1,5,9,…,73,77}, \{1,5,9,\ldots,73,77\},

so

a1+a2+⋯+a18=702. \boxed{a_1+a_2+\cdots+a_{18}=702}.
