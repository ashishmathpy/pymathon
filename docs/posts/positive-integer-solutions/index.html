Introduction

Counting integer solutions is a central idea in algebra, number theory, and combinatorics. Here the word positive matters: each variable must be at least 11, not merely non-negative.

We will count the solutions in four complementary ways. The mathematics supplies the proof; Python provides an independent verification.

Difficulty: Intermediate
Prerequisites: Basic algebra, inequalities, and elementary counting
Estimated reading time: 8 minutes

Problem

Find the number of positive integer solutions of

x+2y+3z=15, x+2y+3z=15,

where x,y,z∈ℤ+x,y,z\in\mathbb Z^+. The answer will be

12. \boxed{12}.

Method 1: Fix One Variable and Count

Solving for xx gives

x=15−2y−3z. x=15-2y-3z.

Since x≥1x\geq1, we have 2y+3z≤142y+3z\leq14. Because y≥1y\geq1, this gives 2+3z≤142+3z\leq14, so z≤4z\leq4. Thus z=1,2,3,4z=1,2,3,4.

For each fixed zz, count the positive values of yy for which x=15−2y−3zx=15-2y-3z stays positive.

Method 2: Convert Positive Integers to Non-negative Integers

Write

x=a+1,y=b+1,z=c+1, x=a+1,\qquad y=b+1,\qquad z=c+1,

where a,b,c≥0a,b,c\geq0. Substitution gives

(a+1)+2(b+1)+3(c+1)=15, (a+1)+2(b+1)+3(c+1)=15,

and hence

a+2b+3c=9. a+2b+3c=9.

For fixed cc, this becomes a+2b=9−3ca+2b=9-3c. An equation a+2b=Na+2b=N, with a,b≥0a,b\geq0, has ⌊N/2⌋+1\left\lfloor N/2\right\rfloor+1 solutions because bb ranges from 00 to ⌊N/2⌋\left\lfloor N/2\right\rfloor.

Method 3: Direct Floor-Function Formula

For fixed positive integer zz, positivity of xx implies

15−2y−3z≥1, 15-2y-3z\geq1,

so

y≤14−3z2. y\leq\frac{14-3z}{2}.

The number of positive integer values of yy is therefore

⌊14−3z2⌋. \left\lfloor\frac{14-3z}{2}\right\rfloor.

Since z=1,2,3,4z=1,2,3,4,

N=∑z=14⌊14−3z2⌋. N=\sum_{z=1}^{4}\left\lfloor\frac{14-3z}{2}\right\rfloor.

Evaluating the sum,

N=⌊112⌋+⌊82⌋+⌊52⌋+⌊22⌋=5+4+2+1=12. N= \left\lfloor\frac{11}{2}\right\rfloor+ \left\lfloor\frac{8}{2}\right\rfloor+ \left\lfloor\frac{5}{2}\right\rfloor+ \left\lfloor\frac{2}{2}\right\rfloor =5+4+2+1=\boxed{12}.

Method 4: Generating Functions

The positive values of xx, 2y2y, and 3z3z are encoded respectively by

t1−t,t21−t2,t31−t3. \frac{t}{1-t},\qquad \frac{t^2}{1-t^2},\qquad \frac{t^3}{1-t^3}.

Their product is

G(t)=(t1−t)(t21−t2)(t31−t3)=t6(1−t)(1−t2)(1−t3). G(t)= \left(\frac{t}{1-t}\right) \left(\frac{t^2}{1-t^2}\right) \left(\frac{t^3}{1-t^3}\right) =\frac{t^6}{(1-t)(1-t^2)(1-t^3)}.

Choosing txt^x, t2yt^{2y}, and t3zt^{3z} from the three factors produces

tx⋅t2y⋅t3z=tx+2y+3z. t^x\cdot t^{2y}\cdot t^{3z}=t^{x+2y+3z}.

Thus every positive solution contributes exactly one t15t^{15} term, so the answer is

[t15]G(t)=[t9]1(1−t)(1−t2)(1−t3). [t^{15}]G(t) =[t^9]\frac{1}{(1-t)(1-t^2)(1-t^3)}.

Expanding only as far as the required term,

1(1−t)(1−t2)(1−t3)=1+t+2t2+3t3+4t4+5t5+7t6+8t7+10t8+12t9+O(t10). \frac{1}{(1-t)(1-t^2)(1-t^3)} =1+t+2t^2+3t^3+4t^4+5t^5+7t^6 +8t^7+10t^8+12t^9+O(t^{10}).

Hence the coefficient of t9t^9 is 1212, in agreement with the direct counts.

Comparison of Methods

Method Main idea Best use
Fix one variable Count directly after fixing zz Small equations and hand calculations
Shift variables Replace positivity by non-negativity Connecting with standard combinatorics
Floor formula Express each row count compactly General formulas
Generating functions Read a coefficient from a product of series Weighted counting problems

Conclusion

The four methods describe the same count from different angles: direct counting lists the cases, shifting variables converts the problem to non-negative solutions, floor functions compress the count, and generating functions provide a systematic method. The Python calculation is a final check, not part of the proof.

12 positive integer solutions. \boxed{12\text{ positive integer solutions}}.

7+5+4+2+1=19. 7+5+4+2+1=\boxed{19}.

NoteSolution for Problem 2

The row counts are 6,4,3,16,4,3,1, giving

6+4+3+1=14. 6+4+3+1=\boxed{14}.

TipHint for Problem 3

Use x=20−3y−4zx=20-3y-4z and count the positive values of yy for each possible positive zz.
