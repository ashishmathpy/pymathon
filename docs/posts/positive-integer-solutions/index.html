Introduction

Counting integer solutions is a central idea in algebra, number theory, and combinatorics. Here the word positive matters: each variable must be at least \(1\), not merely non-negative.

We will count the solutions in four complementary ways. The mathematics supplies the proof; Python provides an independent verification.

Problem

Find the number of positive integer solutions of

\[ x+2y+3z=15, \]

where \(x,y,z\in\mathbb Z^+\). The answer will be

\[ \boxed{12}. \]

Method 1: Fix One Variable and Count

Solving for \(x\) gives

\[ x=15-2y-3z. \]

Since \(x\geq1\), we have \(2y+3z\leq14\). Because \(y\geq1\), this gives \(2+3z\leq14\), so \(z\leq4\). Thus \(z=1,2,3,4\).

For each fixed \(z\), count the positive values of \(y\) for which \(x=15-2y-3z\) stays positive.

Value of \(z\) Equation for \(x\) and \(y\) Allowed values of \(y\) Number of solutions
\(1\) \(x+2y=12\) \(1,2,3,4,5\) \(5\)
\(2\) \(x+2y=9\) \(1,2,3,4\) \(4\)
\(3\) \(x+2y=6\) \(1,2\) \(2\)
\(4\) \(x+2y=3\) \(1\) \(1\)

Therefore,

\[ 5+4+2+1=\boxed{12}. \]

Method 2: Convert Positive Integers to Non-negative Integers

Write

\[ x=a+1,\qquad y=b+1,\qquad z=c+1, \]

where \(a,b,c\geq0\). Substitution gives

\[ (a+1)+2(b+1)+3(c+1)=15, \]

and hence

\[ a+2b+3c=9. \]

For fixed \(c\), this becomes \(a+2b=9-3c\). An equation \(a+2b=N\), with \(a,b\geq0\), has \(\left\lfloor N/2\right\rfloor+1\) solutions because \(b\) ranges from \(0\) to \(\left\lfloor N/2\right\rfloor\).

\(c\) Equation Number of solutions
\(0\) \(a+2b=9\) \(\left\lfloor9/2\right\rfloor+1=5\)
\(1\) \(a+2b=6\) \(\left\lfloor6/2\right\rfloor+1=4\)
\(2\) \(a+2b=3\) \(\left\lfloor3/2\right\rfloor+1=2\)
\(3\) \(a+2b=0\) \(1\)

Again,

\[ 5+4+2+1=\boxed{12}. \]

Method 3: Direct Floor-Function Formula

For fixed positive integer \(z\), positivity of \(x\) implies

\[ 15-2y-3z\geq1, \]

so

\[ y\leq\frac{14-3z}{2}. \]

The number of positive integer values of \(y\) is therefore

\[ \left\lfloor\frac{14-3z}{2}\right\rfloor. \]

Since \(z=1,2,3,4\),

\[ N=\sum_{z=1}^{4}\left\lfloor\frac{14-3z}{2}\right\rfloor. \]

Evaluating the sum,

\[ N= \left\lfloor\frac{11}{2}\right\rfloor+ \left\lfloor\frac{8}{2}\right\rfloor+ \left\lfloor\frac{5}{2}\right\rfloor+ \left\lfloor\frac{2}{2}\right\rfloor =5+4+2+1=\boxed{12}. \]

Method 4: Generating Functions

The positive values of \(x\), \(2y\), and \(3z\) are encoded respectively by

\[ \frac{t}{1-t},\qquad \frac{t^2}{1-t^2},\qquad \frac{t^3}{1-t^3}. \]

Their product is

\[ G(t)= \left(\frac{t}{1-t}\right) \left(\frac{t^2}{1-t^2}\right) \left(\frac{t^3}{1-t^3}\right) =\frac{t^6}{(1-t)(1-t^2)(1-t^3)}. \]

Choosing \(t^x\), \(t^{2y}\), and \(t^{3z}\) from the three factors produces

\[ t^x\cdot t^{2y}\cdot t^{3z}=t^{x+2y+3z}. \]

Thus every positive solution contributes exactly one \(t^{15}\) term, so the answer is

\[ [t^{15}]G(t) =[t^9]\frac{1}{(1-t)(1-t^2)(1-t^3)}. \]

That coefficient is \(12\), in agreement with the direct counts.

Python Verification

The following program uses the equation to compute \(x\) after choosing positive \(y\) and \(z\). It prints the triples in the same order as the hand-counting method.

solutions = []

for z in range(1, 5):
    for y in range(1, 16):
        x = 15 - 2 * y - 3 * z
        if x > 0:
            solution = (x, y, z)
            solutions.append(solution)
            print(solution)

print("Total number of solutions:", len(solutions))

The program confirms the count, but the earlier methods explain why the count is \(12\).

Complete List of Solutions

The complete solution list is

\[ \begin{aligned} z=1:&\quad (10,1,1),\ (8,2,1),\ (6,3,1),\ (4,4,1),\ (2,5,1),\\ z=2:&\quad (7,1,2),\ (5,2,2),\ (3,3,2),\ (1,4,2),\\ z=3:&\quad (4,1,3),\ (2,2,3),\\ z=4:&\quad (1,1,4). \end{aligned} \]

Every listed triple satisfies \(x+2y+3z=15\), and the list contains \(5+4+2+1=12\) triples.

Comparison of Methods

Method Main idea Best use
Fix one variable Count directly after fixing \(z\) Small equations and hand calculations
Shift variables Replace positivity by non-negativity Connecting with standard combinatorics
Floor formula Express each row count compactly General formulas
Generating functions Read a coefficient from a product of series Weighted counting problems

Conclusion

The four methods illuminate the same result from different angles: direct counting is concrete, shifting variables connects the problem to non-negative solutions, floor functions compress the count, and generating functions give a powerful general framework. Python verifies the final list.

\[ \boxed{12\text{ positive integer solutions}}. \]
