# Bundled webfonts

`Euler-Math.woff2` is the webfont build of Euler Math, the OpenType
continuation of Hermann Zapf's Euler mathematics font.

- Project: https://ctan.org/pkg/euler-math
- Webfont build: https://mathfonts.github.io/Euler/
- License: SIL Open Font License 1.1

`JetBrainsMono-Regular.woff2` is the regular webfont from JetBrains Mono,
a typeface designed specifically for source code.

- Project: https://www.jetbrains.com/lp/mono/
- Source: https://github.com/JetBrains/JetBrainsMono
- License: SIL Open Font License 1.1
